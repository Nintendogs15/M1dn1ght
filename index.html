❓ Help & Information

About M1dn1ght

M1dn1ght is a web-based tool for managing ChromeOS systems. It allows you to uninstall Chrome extensions and force unenroll enterprise-managed Chromebooks from your organization's management servers.

🔍 How to Find Your Extension ID

  1. Open Chrome on your Chromebook
  2. Go to chrome://extensions/
  3. Enable Developer mode (toggle in top right)
  4. Find your extension in the list
  5. Copy the ID displayed below the extension name

💾 System Paths Affected by Unenrollment

  • /etc/opt/google/chrome/policies/ - Enterprise policies
  • /var/lib/whitelist/ - Whitelist configurations
  • /var/lib/devicesettings/ - Device settings
  • /var/lib/dbus/ - Device state
  • /var/lib/tpm/ - TPM enterprise data

⚠️ Important Notes

  • Personal Use Only: Only run on your own Chromebook with explicit permission
  • Irreversible: Cannot be undone without re-enrollment through IT administrator
  • Data Loss Risk: Some data may be lost during unenrollment
  • Legal Consequences: Unauthorized unenrollment may violate employment agreements

✅ After Unenrollment

  • Device will reboot automatically
  • You'll have full control over your Chromebook
  • Enterprise monitoring will cease
  • You can install any extensions or applications

🔧 Troubleshooting

  • Extension Not Found: Verify the extension ID is correct
  • Permission Denied: Ensure you have owner account access
  • Unenrollment Failed: Check that Linux container has proper permissions
  • Device Won't Boot: Force shutdown and restart, or use Recovery Mode (Ctrl+Alt+Shift+R)
