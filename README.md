# M1dn1ght

A Chromebook enterprise enrollment removal tool.

## Description

M1dn1ght is software designed to help users unenroll their Chromebooks from enterprise management policies. This tool provides both a command-line interface and a web-based interface for managing Chromebook enrollment status.

## Features

- Remove enterprise enrollment from Chromebooks
- User-friendly web interface
- Command-line tools for advanced users
- Support for multiple Chromebook models

## Getting Started

### Web Interface
Simply open `index.html` in your browser to access the web-based unenrollment tool.

### Command Line
Use the provided CLI tools for programmatic unenrollment.

## Installation

1. Clone the repository
2. Follow the build instructions in `BUILD.md`
3. Run the unenrollment tool
